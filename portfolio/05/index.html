AluraTube

+ informações

Esse projeto foi desenvolvido durante uma imersão da Alura, com duração de 5 dias. É um repositório de links do YouTube mas com uma interface muito bem acabada. Permite adicionar vídeos usando o Supabase, que é um Backend-as-a-Service (BaaS), e rodando a partir do Vercel. Tecnologias usadas: HTML5, CSS3, Node.js para introdução e instalação das bibliotecas/frameworks, React.js, Next.js, Styled-Components e as ferramanetas VSCode e Figma. Outros serviços foram usados como o Vercel para o deploy e o Supabase que fornece uma base de dados online. Sim, você pode ver o projeto funcionando, basta clicar em "+ informações" acima. =}
