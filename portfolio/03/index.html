LanchesMac

+ informações

LanchesMac é um projeto de curso, focado em Back-End e Microsoft MVC, bastante complexo, além de demonstrar várias ferramentas do padrão MVC (Model View Controller). Aqui é aprendido os fundamentos da ASP .NET Core MVC, Entity Framework Core; Data Annotations e Migrations e modelo padrão Repository. As ferramentas usadas foram VisualStudioCode e a infraestrutura Nuget, bem como o Microsoft SQL Server. Você pode ver o código fonte do projeto clicando no botão "+ informações" acima.
